// MISSION OBJECTIVE

Project ini bukan hanya project mid Web design saya saja, tetapi juga sebagai latihan saya untuk membuat Website yang mencakup sedikit UI/UX dan juga prompting AI. Saya terinspirasi dari sebuah website yang saya lihat dari cssAwards yang bernama "Ledger". Yang dimana temanya kebanyakan warna putih dan hitam dan juga kotak, lalu ada seperti HUD yang membuat tampilannya menjadi unik dan keren. Project ini dibangun sebagai Project untuk mata kuliah Desain Web yang dibimbing oleh Sir Regi Najoan. Tujuannya adalah untuk tidak hanya menambah sebuah portofolio, tetapi juga untuk mengeksplorasi bagaimana sebuah tema visual yang kuat dapat diterapkan secara konsisten di berbagai halaman untuk menciptakan pengalaman pengguna yang imersif.

// SYSTEM SPECS

  • Core: HTML5, CSS3, JavaScript (ES6)
  • Framework: Bootstrap 5
  • Storage: LocalStorage API
  • Assets: Google Fonts, SweetAlert2
  • AI: Google Gemini, Deepseek, ChatGPT

// DEVELOPMENT LOG

Proyek ini berevolusi dari satu halaman login menjadi sebuah situs mini yang fungsional. Fokus utama adalah pada konsistensi komponen UI, mulai dari navigasi HUD, tombol, hingga formulir, untuk memastikan setiap interaksi terasa familiar bagi pengguna di seluruh situs.
